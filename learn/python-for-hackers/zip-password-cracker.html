What is a ZIP Password Cracker?

A ZIP Password Cracker is a Python script or tool that helps you find the forgotten password of a ZIP file. If a ZIP file is protected with AES encryption, and you legally own it, you can try to recover the password using this method.

Why Use a ZIP Cracker?

If you've locked your own ZIP file and can't remember the password, you can try common passwords from a wordlist to recover it. This is known as a "dictionary attack."

Common Attack Methods

  • Brute Force: Try all possible combinations (slow but effective).
  • Dictionary Attack: Try common passwords from a list.
  • Mask Attack: Try based on known patterns like "admin***" or "1234****".

Python Script for AES ZIP Cracking

The Python script below uses the pyzipper module to check if passwords from a wordlist can unlock an AES-encrypted ZIP file. 

# Import the pyzipper module to handle AES-encrypted ZIP files
import pyzipper

# Import os module to work with the file system
import os

# Define a function to crack an AES ZIP file using a wordlist
def crack_aes_zip(zip_file_path, wordlist_path):
    # Step 1: Make sure the ZIP file exists
    if not os.path.exists(zip_file_path):
        print(f"[x] ZIP file not found: {zip_file_path}")
        return  # Exit the function if ZIP doesn't exist

    # Step 2: Check if the wordlist file exists
    if not os.path.exists(wordlist_path):
        print(f"[x] Wordlist file not found: {wordlist_path}")
        return  # Exit if the wordlist file is missing

    # Step 3: Read all passwords from the wordlist into a list
    with open(wordlist_path, 'r', encoding='utf-8', errors='ignore') as f:
        # Strip whitespace and skip blank lines
        passwords = [line.strip() for line in f if line.strip()]

    try:
        # Step 4: Open the ZIP file using pyzipper with AES decryption support
        with pyzipper.AESZipFile(zip_file_path) as zf:
            # Get a list of files inside the ZIP
            file_list = zf.namelist()

            # If ZIP has no files or is unreadable, exit
            if not file_list:
                print("[!] ZIP file is empty or has no readable entries.")
                return

            # Step 5: Try each password one by one from the wordlist
            for pwd in passwords:
                try:
                    # Set the password (must be bytes)
                    zf.pwd = pwd.encode('utf-8')

                    # Try opening the first file in the archive
                    with zf.open(file_list[0]) as f:
                        f.read(10)  # Read 10 bytes to test the password

                    # If no error occurs, password is correct
                    print(f"[+] Password found: {pwd}")
                    return True  # Stop after finding the right password

                except (RuntimeError, pyzipper.BadZipFile, ValueError):
                    # If password is wrong, continue trying
                    print(f"[-] Tried: {pwd}")
                except KeyboardInterrupt:
                    # If the user manually stops the script
                    print("[!] Interrupted by user.")
                    return

            # If all passwords failed
            print("[!] Password not found in wordlist.")
            return False

    except pyzipper.BadZipFile:
        # If the file is not a valid AES ZIP archive
        print("[x] Not a valid AES-encrypted ZIP file.")
        return

# === Start of the script ===
if __name__ == "__main__":
    # Ask user for ZIP file path
    zip_path = input("Enter path to AES-encrypted ZIP file: ").strip()

    # Ask user for the path to the password wordlist
    wordlist_path = input("Enter path to your password wordlist: ").strip()

    # Run the function with user inputs
    crack_aes_zip(zip_path, wordlist_path)

Line-by-Line Code Explanation

  • import pyzipper: A module that allows opening AES-encrypted ZIP files.
  • import os: Used to check if files exist before using them.
  • crack_aes_zip(...): Main function that attempts to find the password.
  • os.path.exists(...): Checks if ZIP and wordlist files are present.
  • with open(...): Reads passwords from the wordlist, removing blank lines.
  • pyzipper.AESZipFile(...): Opens the ZIP file using AES support.
  • namelist(): Gets list of files inside the ZIP archive.
  • for pwd in passwords: Tries each password one by one.
  • zf.pwd = pwd.encode(...): Converts the password to byte format.
  • zf.open(...).read(): Tries to read a few bytes using that password.
  • if success: The password is printed and process stops.
  • if fail: It prints that password was incorrect and continues.
