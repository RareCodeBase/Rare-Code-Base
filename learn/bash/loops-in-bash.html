Loops in Bash Scripting (for, while, until)

A loop lets you run a block of code again and again. In Bash scripting, we mostly use:

  • for loop – Repeats for a fixed number of times
  • while loop – Repeats as long as the condition is true
  • until loop – Repeats until the condition becomes true

1. For Loop (Fixed Repetition)

#!/bin/bash

# Print numbers from 1 to 5
for i in {1..5}
do
  echo "Number: $i"
done

Explanation:

  • for i in {1..5} runs the loop 5 times
  • $i takes values from 1 to 5
  • done is used to end the loop

2. For Loop with List

#!/bin/bash

# Loop through names
for name in Alice Bob Charlie
do
  echo "Hello, $name"
done

Explanation:

  • This loop runs 3 times (one for each name)
  • $name takes each value from the list

3. While Loop (Condition Based)

#!/bin/bash

count=1

# Repeat while count is less than or equal to 5
while [ $count -le 5 ]
do
  echo "Count: $count"
  ((count++)) # Increase count by 1
done

Explanation:

  • while [ $count -le 5 ] keeps running as long as the condition is true
  • ((count++)) increases count by 1

4. Until Loop (Opposite of While)

#!/bin/bash

n=1

# Run until n becomes greater than 5
until [ $n -gt 5 ]
do
  echo "n = $n"
  ((n++))
done

Explanation:

  • until keeps running while the condition is false
  • Here it runs until n is greater than 5

5. Looping Through Files

#!/bin/bash

# List all .txt files in the current directory
for file in *.txt
do
  echo "Found file: $file"
done

Explanation:

  • *.txt matches all files ending with .txt
  • $file holds each filename in the loop

6. Infinite Loop (Be Careful!)

#!/bin/bash

# Infinite loop example (press Ctrl+C to stop)
while true
do
  echo "This will keep running..."
  sleep 1
done

Explanation:

  • while true never becomes false, so it loops forever
  • sleep 1 pauses for 1 second between each loop

Tips for Beginners

  • Always close loops with done
  • Use break to stop a loop early
  • Use continue to skip current step and move to next

Example with break & continue

#!/bin/bash

for i in {1..10}
do
  if [ $i -eq 3 ]; then
    echo "Skipping 3"
    continue
  fi

  if [ $i -eq 8 ]; then
    echo "Stopping at 8"
    break
  fi

  echo "i = $i"
done

Explanation:

  • continue skips step 3
  • break stops the loop when i = 8

Conclusion

Loops are a key part of Bash scripting. They help repeat tasks, go through lists, and process files. Practice using them in real tasks to get comfortable.
