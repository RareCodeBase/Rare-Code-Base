Functions in Bash Scripting

A function in Bash is a block of code you can reuse. It helps avoid repeating the same code again and again. You can call a function anytime in your script after defining it.

Why Use Functions?

  • Organize your script better
  • Make code easier to read and maintain
  • Reuse the same code many times

1. Simple Function Example

#!/bin/bash

# Define a function named greet
greet() {
  echo "Hello! Welcome to Bash scripting."
}

# Call (run) the function
greet

Explanation:

  • greet() is the function name
  • Code inside { } runs when the function is called
  • greet at the end calls the function

2. Function with Arguments

#!/bin/bash

# Function that takes two arguments
say_hello() {
  echo "Hello, $1!"
  echo "Your role is: $2"
}

# Call the function with two arguments
say_hello "Alice" "Developer"

Explanation:

  • $1 is the first argument ("Alice")
  • $2 is the second argument ("Developer")
  • You can pass any values while calling the function

3. Function with Return Value

#!/bin/bash

# Function that returns the sum of two numbers
add_numbers() {
  local sum=$(( $1 + $2 ))  # Add arguments
  return $sum               # Return result
}

add_numbers 4 6             # Call function
result=$?                   # Capture the returned value
echo "Sum is: $result"

Explanation:

  • return can only return numbers (0-255)
  • $? holds the last return value
  • local keeps sum variable inside the function

Note: Use echo to return text values instead of return.

4. Return Text from Function

#!/bin/bash

# Function to return full name
full_name() {
  echo "$1 $2"
}

# Store result in a variable
name=$(full_name "John" "Doe")
echo "Your full name is: $name"

Explanation:

  • $(...) captures the output of the function
  • Useful for returning strings

5. Check If Function Exists

#!/bin/bash

# Check if a function is already defined
type greet &>/dev/null

if [ $? -eq 0 ]; then
  echo "Function 'greet' exists."
else
  echo "Function 'greet' not found."
fi

Explanation:

  • type checks if a function/command exists
  • $? tells whether it was found

Best Practices

  • Use lowercase names for functions: my_function
  • Keep functions short and simple
  • Put function definitions at the top of your script
  • Use comments to explain what each function does

Conclusion

Functions are very useful in Bash scripting. They help you write clean, reusable, and organized code. Practice writing your own functions and using them in real scripts.
